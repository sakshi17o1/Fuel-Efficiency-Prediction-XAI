# Explainable AI-Driven Fuel Efficiency Prediction
This repository contains the implementation of the research paper:
"Explainable AI-Driven Fuel Efficiency Prediction Using Ensemble Learning and Vehicle Configuration Data"

## Files
- Fuel_Efficiency_Prediction.ipynb
- requirements.txt
- auto-mpg.csv
- archive.zip

## Models
- Random Forest
- XGBoost
- Hybrid Ensemble

## Explainability
- SHAP
  
## Scope Note

This repository contains the complete explainability pipeline developed for 
this project, including both SHAP and LIME-based analyses.

The manuscript "Explainable AI-Driven Fuel Efficiency Prediction Using 
Ensemble Learning and Vehicle Configuration Data" (Scientific Reports) 
reports results using the SHAP component only (see Part C / Section 8.3 
of the notebook, and Section 8.3 of the manuscript).

The SHAP-vs-LIME comparison (Part C2, C3 of the notebook) supports a 
separate, forthcoming study on explainability-method comparison and is 
not part of this manuscript's reported results.
